# Local Turístico

Página de destino turístico desenvolvida durante os estudos.

![Preview](.github/preview.png)

## Sobre

Projeto de uma página web apresentando os principais pontos turísticos de Busan, Coreia do Sul. A página destaca 3 destinos imperdíveis para amantes da história, incluindo templos budistas e parques com vistas panorâmicas.

## Tecnologias

- HTML
- CSS

